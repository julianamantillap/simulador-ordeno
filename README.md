# Simulador de proceso de ordeño - Modelo M/G/1

Este proyecto implementa un simulador del proceso de ordeño de una finca, modelado mediante teoría de colas bajo el enfoque M/G/1.

El sistema real cuenta con 8 puestos de ordeño, pero para efectos del análisis se representa como una cola única con un servidor equivalente.

El simulador permite analizar el comportamiento del sistema y calcular métricas como:

- Tasa de llegada (λ)
- Tasa de servicio (μ)
- Utilización (ρ)
- Longitud de cola (Lq)
- Tiempo de espera (Wq)
- Tiempo total en el sistema (W)

## Tecnologías

- React
- TypeScript
- Vite
- Three.js
- XLSX
